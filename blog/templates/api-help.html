API

How to use the API?

Methods:

Posts:
To receive an API of all posts, write or click: http://127.0.0.1:8000/api/posts/blog/?format=json
To receive an API of one of the posts, write:
* where (post_id) is an id of the specific post
http://127.0.0.1:8000/api/posts/blog/(post_id)/?format=json
Users:
To receive an API of all users, write or click: http://127.0.0.1:8000/api/profiles/profile/?format=json
To receive an API of one of the users, write:
* where (user_id) is an id of the specific user
http://127.0.0.1:8000/api/profiles/profile/(user_id)/?format=json
